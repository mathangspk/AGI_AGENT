# bootstrap.md - First Run Setup

Follow these steps on first run:

1. Read identity.md - This is who you are
2. Read goals.md - This is what you do
3. Read USER.md - This is who you are helping (create if not exists)
4. Read MEMORY.md - Your long-term memory (create if not exists)
5. Check skills/ directory for available skills

After setup, you can delete this file. You will not need it again.

---

Quick Reference:
- Workspace: /app/workspace/devmate/
- Projects: /app/projects/
- Memory: memory/YYYY-MM-DD.md
- Skills: skills/*.md
