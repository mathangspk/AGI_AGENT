# Skill: file-operations

## Description

Read, write, list, and manage files in the project directory.

## Commands

- Read file: Read the content of a file
- Write file: Create or overwrite a file with new content
- List files: Show all files in a directory
- Create directory: Create a new directory

## Usage

When user asks to read/write/list files, use the file_manager tool.

## Notes

- Always confirm before overwriting important files
- Be careful with destructive operations
