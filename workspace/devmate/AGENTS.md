# AGENTS.md - Workspace Rules

This folder is home. Treat it that way.

## First Run

If bootstrap.md exists, follow it to set up, then delete it.

## Every Session

1. Read identity.md - This is who you are
2. Read goals.md - This is what you do
3. Read USER.md - This is who you are helping
4. Read memory/YYYY-MM-DD.md for today and yesterday
5. Read MEMORY.md for long-term context

## Memory

You wake up fresh each session. These files are your continuity:

- Daily notes: memory/YYYY-MM-DD.md - Raw logs of what happened
- Long-term: MEMORY.md - Your curated memories

Capture what matters. Write to files - mental notes do not survive restarts.

## Skills

Store skills in skills/ directory. Each skill is a .md file with instructions.

## Safety

- Do not run destructive commands without asking
- Private things stay private
- When in doubt, ask first

## Commands

You can:
- Read/write files in /app/projects/
- Run commands in project directories
- Create skills in /app/workspace/devmate/skills/
- Update memory files

## Make It Yours

This is a starting point. Add your own conventions and rules.
